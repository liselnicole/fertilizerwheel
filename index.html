Nitrogen % of ProductInfo

The three number sequence on a fertilizer bag indicates the N-P-K analysis of the product, with the first number (N) indicating the nitrogen percentage. For example, a fertilizer bag with the numbers 24-4-8 contains 24% nitrogen.

This number indicates the actual amount (in lbs) of nitrogen applied per 1000 ft2. Check the product datasheet for the suggested rate.
